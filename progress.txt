## Codebase Patterns
- Types are in src/types/index.ts — import from there in all utilities and components
- Vitest is used for testing (not Jest), configured in vitest.config.ts
- Each component lives in its own directory under src/components/ with a co-located .css file
- No inline styles allowed — all styles go in .css files
